# Section Intro

Degenerative AI fundamentals and Amazon Bedrock, the foundation for everything you'll build in this course. Before we dive into advanced architectures and exam-level scenarios, this section will make sure you're fully grounded in the core GenAI concepts and how they're implemented on AWS. We'll start the basics: how Amazon Bedrock works, how to explore models in the Bedrock playground, and how to fine-tune foundation models using techniques like LoRA. You'll also learn how Bedrock supports multimodal models and pipelines, combining text, images, and more. Next, we move into retrieval augmented generation, vector stores, and vector knowledge bases. You'll get hands-on with chunking strategies, embedding optimization, and pre-retrieval techniques, everything you need to build high-quality rag systems in production. We'll also cover bedrock guardrails, token level redaction, prompt engineering, and the anatomy of an effective prompt. You'll learn prompt best practices, types of prompts, how to mitigate misuse and bias, and how to structure prompts consistently with prompt flows and response format templates. Finally, we'll look at enterprise integration patterns and the AWS Well-Architected Generative AI lens so you understand how GenAI systems fit into secure, scalable, real-world architectures. By the end of this section, you'll have a solid foundation in GenAI and Amazon Bedrock, ready to build real applications and ready for the deeper exam-focused material ahead. Let's get started.
